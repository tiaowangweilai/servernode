# agv_bridge

## 🌉 功能描述
`agv_bridge` 是系统的核心调度节点，采用**中介者模式**在远程控制端（Unity/Web）与 ROS 2 系统之间建立通信桥梁。

## 🛠️ 核心功能
- **消息分发**: 将接收到的 WebSocket JSON 指令解析并路由到底盘、升降或机械臂处理器。
- **状态汇总**: 收集各子系统的 ROS 2 状态并统一上报给上位机。
- **设备管理**: 抽象机器人及其组成的各个设备组件。
